# Secret Manager Frontend

Next.js 15 frontend with App Router, Tailwind CSS v4, and shadcn/ui.

## Getting Started

```bash
# Install dependencies
npm install

# Copy env file
cp .env.example .env.local

# Run dev server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS v4
- shadcn/ui
- NextAuth.js (Auth.js v5)
- SWR for data fetching
